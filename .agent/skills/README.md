# AI Skills

This directory contains specialized skills for the AI coding assistant.

## Structure
Each skill should be in its own subdirectory:
- `SKILL.md`: Main instruction file (Required)
- `scripts/`: Helper scripts (Optional)
- `examples/`: Reference implementations (Optional)
- `resources/`: Additional assets (Optional)

## How to add a skill
1. Create a new folder in `.agent/skills/`.
2. Copy `template/SKILL.md` to your new folder.
3. Fill in the YAML frontmatter and instructions.
